VRChat Booth Manager v0.2.0 (ベータ版)
========================================

🎯 概要
VRChat改変向けBooth商品管理ツール
アバターと商品を効率的に管理できるデスクトップアプリケーションです。

📋 システム要件
- Windows 10/11 (64bit)
- Node.js 18以上 (自動でインストールされます)
- インターネット接続 (サムネイル自動取得時)

🚀 起動方法
1. "VRChat-Booth-Manager.bat" をダブルクリック
2. 初回起動時は依存関係のインストールが行われます
3. しばらく待つとアプリケーションが起動します

✨ 主な機能
- アバター管理: 所有アバターの登録・管理
- 商品管理: Booth商品の登録・分類・検索
- サムネイル自動取得: Booth URLから画像を自動取得
- フィルタリング: アバター・カテゴリ別の絞り込み
- Unity連携: unitypackageファイルの直接起動
- データ管理: エクスポート・インポート機能

⌨️ ショートカット
- Ctrl+N: 新しい商品を追加
- Ctrl+A: 新しいアバターを追加
- Ctrl+E: データをエクスポート
- Ctrl+I: データをインポート
- Ctrl+,: 設定画面を開く
- Escape: モーダルを閉じる

🔧 使い方
1. 「アバター追加」で所有アバターを登録
2. 「商品を追加」でBooth商品を登録
3. Booth URLを入力すると自動でサムネイル取得
4. 対応アバターを選択して関連付け
5. フィルター機能で商品を整理・検索

💾 データについて
- データはローカルに保存されます
- 設定画面からデータのエクスポート・インポートが可能
- 定期的なバックアップを推奨します

❓ トラブルシューティング
- 起動しない場合: 管理者権限で実行してみてください
- サムネイル取得失敗: 手動でURLを入力してください
- ファイルが開けない: パスが正しいか確認してください
- データが消えた: エクスポートファイルからインポートしてください
